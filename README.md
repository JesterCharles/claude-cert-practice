# CCA-F Cert Practice Tracker

Team prep hub for the **Claude Certified Architect — Foundations** exam.

**Target sitting:** Thursday, May 29, 2026.

## 🎯 Practice Exam — start early, test often

Don't save practice questions for the diagnostic. Quick reps any slow weekday compound; the May 26 diagnostic should confirm what you already know, not surprise you.

- **[Vercel mock exam](https://claude-certified-architect-mock-exam-cyberskill.vercel.app/)** — free, no login, best for warm-up reps.
- **[claudecertifications.com](https://claudecertifications.com/)** — 25 questions, public.
- **[Exam Guide PDF](https://everpath-course-content.s3-accelerate.amazonaws.com/instructor%2F8lsy243ftffjjy1cx9lm3o2bw%2Fpublic%2F1773274827%2FClaude+Certified+Architect+%E2%80%93+Foundations+Certification+Exam+Guide.pdf)** — 12 sample questions inside, plus the canonical domain weights.

> Note: Anthropic's Skilljar portal hosts the cert registration and the free Academy courses, but **does not provide a separate practice exam**. The May 26 timed diagnostic uses Vercel + claudecertifications.com.

## How to use this repo

1. Read the [Exam Guide PDF](./00-start-here/resources.md#tier-1--authoritative-start-here) end-to-end on day 1 (May 12). Don't skip this — domain weights and scenario coverage are in here.
2. Follow the [daily plan](./00-start-here/study-plan.md) — core column every day, stretch when you have time.
3. Hit each checkpoint at the end of weeks 1 and 2. If you fail a checkpoint, slow down before moving on.
4. Take the diagnostic exam on May 26. The red-flag rule at the bottom of the study plan applies.

## Tracker

There's a live Cowork artifact (`ccaf-tracker`) that turns the daily plan into an interactive checklist with progress bars and an exam-day countdown. Each teammate's checkmarks live in their own browser (localStorage), so it's a personal tracker per person — but the underlying plan is shared.

## Conventions

- All dates are 2026.
- "Domain" refers to the five exam domains as named in the Exam Guide PDF (D1 Agentic, D2 Claude Code, D3 MCP, D4 Prompt, D5 Context).
- If you change the plan or resources, update both the markdown and the tracker artifact so they stay in sync.
